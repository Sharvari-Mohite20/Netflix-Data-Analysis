# Netflix Data Analysis

## About This Project

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles Dataset using Python.

Using the Netflix Titles dataset, I explored trends related to content types, ratings, genres, countries, and content growth over time.

The goal of this project was to practice:
- Data cleaning
- Data exploration
- Data visualization
- Extract insights from real-world data

## Dataset

Netflix Movies and TV Shows Dataset

Source: https://www.kaggle.com/datasets/shivamb/netflix-shows

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## What I Analyzed

- Movies vs TV Shows on Netflix
- Top countries producing Netflix content
- Most common content ratings
- Popular genres
- Content added over the years

## Key Findings

- Netflix has more Movies than TV Shows.
- The United States contributes the highest number of titles.
- TV-MA is one of the most common content ratings.
- Drama and International content appear frequently in the dataset.
